# Mandelbrot set

Mandelbrot set in WebGL!

## Controls

**w**, **a**, **s**, **d** for moving around

**q** and **e** to rotate

**z**/**f**/**shift** to zoom in

**x**/**g**/**control** to zoom out

## Math

### Complex number

z = a + bi

c = x + yi

### Power of 2

f(z) = z^2 + c

(a+bi)^2 = a^2 + 2abi - b^2

real = a^2 - b^2 + x

imaginary = 2abi + yi
